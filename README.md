# 📘 Distance Learning — Infinitive Explorer

A premium, interactive web application designed to help students master English infinitives through immersive storytelling and gamified progress tracking.

## ✨ Features

- **Interactive Storytelling**: Students find and click on infinitives within passages to reveal their meaning and usage.
- **Gamified Progress**: Earn XP and maintain streaks for every infinitive discovered.
- **Smart Teacher Panel**: Teachers can easily add new passages, highlight infinitives using the "Smart Highlighter" tool, and provide detailed explanations.
- **Dual-Mode Experience**: Toggle between Student and Teacher views seamlessly.
- **Professional UI**: Fully responsive design with **Royal & Amber** premium palette and Dark/Light theme support.

## 🚀 Tech Stack

- **React** (v18+)
- **TypeScript**
- **Vite** (Fast build tool)
- **Lucide React** (Modern iconography)
- **CSS3** (Custom design system with HSL variables)

## 🛠 Project Structure

```text
src/
  ├── components/
  │   ├── common/      # Reusable UI components (XPBar, PassageCard, etc.)
  │   ├── student/     # Student-facing dashboards
  │   └── teacher/     # Content management & highlighter tools
  ├── data/            # Static content and initial state
  ├── types/           # Global TypeScript definitions
  ├── utils/           # Shared logic and text parsing
  └── App.tsx          # Main orchestrator
```
